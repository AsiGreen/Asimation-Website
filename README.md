# Asimation - Data & Automation Agency

A modern, animated website for Asimation, a cutting-edge data and automation agency.

## Features

- **Responsive Design** - Works seamlessly on all devices
- **Smooth Animations** - CSS animations and scroll-triggered effects
- **Interactive Elements** - Hover effects, animated counters, and form handling
- **Modern UI** - Clean black, white, and green color scheme
- **Performance Optimized** - Lightweight and fast-loading

## Sections

- **Hero** - Eye-catching landing with animated particles
- **Services** - 6 core service offerings with animated icons
- **Industry Solutions** - Tailored solutions for different sectors
- **Statistics** - Animated counters showcasing achievements
- **About** - Company information with data visualization
- **Contact** - Working contact form

## Technologies

- HTML5
- CSS3 (Animations, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

## Getting Started

Simply open `index.html` in your browser or serve with any static server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then navigate to `http://localhost:8000`
